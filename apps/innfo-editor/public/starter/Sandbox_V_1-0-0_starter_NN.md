# _NN Guide
* _NN Guide: Hello and welcome!
  This is an **element** — the basic building block of an iNNfo model.
  Elements live inside **concepts** (categories that group related things).

  Look at the **left sidebar** → you'll see a tree:
  - Your model root is at the top
  - "Guide" is a **concept** (shown with a colored dot)
  - Inside it, **this element** (and the ones below it)

  Try clicking "Guide" in the tree to see the concept itself.

* _NN Guide: The tree on the left
  The **left sidebar** shows every concept and element as a tree:
  - 📁 Concepts are like folders — they group elements
  - 📄 Elements are the actual items inside them
  - Click any item to select it and see its content here
  - The arrow ▶ lets you expand/collapse a concept

  This model has only one concept: "Guide". In a real model you'd have many.

* _NN Guide: Editor views
  Above this text you'll see tabs: **Sheet** · **Raw** · **Tree** · **Graph**
  - **Sheet** (current): structured form editor
  - **Raw**: plain Markdown source code
  - **Tree**: hierarchical JSON view
  - **Graph**: visual node diagram

  Switch between them — all show the same model, just different views.
  The Graph view is especially useful to see relationships at a glance!
